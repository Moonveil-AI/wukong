# Changelog

All notable changes to @wukong/llm-anthropic will be documented in this file.

## [0.1.0] - 2025-11-13

### Added

- Initial release of Claude LLM adapter
- Support for Claude Sonnet 4.5 and Haiku 4.5
- Streaming responses
- 200K context window support
- Function calling capabilities
- Vision support (Sonnet only)
- Automatic retries with exponential backoff
- Rate limit handling
- Token counting (estimation)
- Comprehensive test suite
- Full documentation
